# RFP TrustRoom Submission Assets

这些文件用于 lablab.ai 提交表单和录屏准备。它们只使用 public-safe replay / mock 证据，不包含 `.env`、Agent keys、真实 room id、真实 message id、私密报告或 `pilotdeck/` 内容。

## Files

- `rfp-trustroom-cover.png`：16:9 cover image，1920x1080。
- `rfp-trustroom-submission-deck.pdf`：8 页 slide presentation PDF。
- `rfp-trustroom-submission-deck.pptx`：可编辑 PowerPoint 版本。
- `video-script-shot-list.md`：5 分钟 video script / shot list。
- `submission-copy.md`：Project Title、Short Description、Long Description 和 tags。
- `dashboard-replay-first-screen.png`、`run-trace-route.png`、`representative-traces.png`：用于 cover/deck 的 public-safe UI 截图素材。

## Boundaries

- Cover / slides / script 可以展示 replay route、Run Trace、Agent Handoff Chain、Representative Item Traces、Q-006 Blocked Impact Path 和 Final Pack。
- 这些资产不能被描述为 production deployment、formal audit、compliance certification、legal advice 或 complete autonomous live Band workflow。
- Public GitHub Repository、Render service、Application URL 和公开视频 URL 仍需要单独完成和 smoke。
